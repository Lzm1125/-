# 校园闲置物品交易平台
基于 Flask 开发的校园闲置交易小项目，实现核心交易功能。

## 核心功能
1. 用户登录
2. 商品发布
3. 商品列表查看
4. 商品价格筛选搜索

## 运行方法
1. 安装依赖：pip install flask
2. 启动项目：python app.py
3. 访问地址：http://127.0.0.1:5000

## 页面访问
- 登录页：http://127.0.0.1:5000/login
- 发布页：http://127.0.0.1:5000/publish
- 商品列表：http://127.0.0.1:5000/product-list

## 小组分工
- 前端：负责所有页面搭建与交互
- 后端：负责接口开发与数据库配置
- 组长：负责任务分配、代码整合与文档编写
